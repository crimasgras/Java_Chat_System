## Java Chat Application

A client-server chat application built with Java Sockets and multithreading,
allowing multiple clients to communicate in real time.

## Technologies
Java, TCP Sockets, Multithreading, Swing GUI, SHA-256 password hashing

## Features
- Real-time messaging between multiple clients
- User authentication with SHA-256 hashing and salt
- Swing GUI with emoji support
- Chat commands (/list, /history, /help)
- Automatic message logging with timestamps

## How to Run
1. Run Server.java
2. Run Client.java for each user
3. Enter credentials and start chatting
